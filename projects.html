  • URL Saver - Chrome Extension

    This is a URL saver as a Chrome Extension. I coded this using Javascript, HTML, and some CSS. Learning how to use localStorage as a pseudo back-end to store key-value pairs and return them to the user. As well as practicing HTML forms, CSS transforms/transitions, and getting the DOM in Javascript.

    Instructions:
    Download the ZIP from GitHub, unzip to a folder and open chrome. Navigate to "chrome://extensions", in the top left click "Load Unpacked" and navigate to the unzipped folder, click "Select folder" and the extension is installed. To use the extension look to the right of the search bar for the puzzle icon. Click it for the dropdown menu that shows all your current extensions that are installed.

    Resources:

    > YouTube - freeCodeCamp.org - Full ~ 8-hour Intro to Javascript Course < > GitHub Respository <
